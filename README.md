M-Verus is an open source android mobile app designed to look for VRSC mining statistics in verus.farm.

## Features
- [x] Global Stats 
- [x] Personal farming stats
- [x] Material 3 colorful themes
- [x] Dark / Light mode
      
## Platforms Support
- [x] Android
- [ ] iOS (less probable)
- [ ] Web (useless)
- [ ] Linux / Windows (useless too)
